Fundamentals of HTML

<a> - Anchor Element

The anchor tag in HTML is used to create hyperlinks, allowing users to navigate to different web pages or resources by clicking on the linked text or image.

<head> - Head Element

The head element in HTML is used to contain meta-information about the document, including metadata, links to external resources like stylesheets and scripts, character set declarations, and other elements that provide information about the document rather than its content visible to users.

target="" - Target Attribute

The target attribute in HTML is used within the anchor element to specify where the linked document should be opened, such as in a new browser window or tab.

# - Link to a Different Part of the Page

The anchor element can create hyperlinks to different parts of the same HTML document using the href attribute to point to the desired location with # followed by the id of the element to link to.

<head> - HTML Element

The html element, the root of an HTML document, should be added after the !DOCTYPE declaration. All content/structure for an HTML document should be contained between the opening and closing html tags.

<!-- --> - Comments

Content inside of comments will not be rendered by browsers, and are usually used to describe a part of code or provide other details. Comments can span single or multiple lines.

<title> - Title Element

The title element in HTML is used to define the title of the document, which appears in the browser's title bar or tab, providing a brief and descriptive label for the webpage.
